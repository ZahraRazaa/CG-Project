# Paperpillar Website Clone

## Overview
This project is a front-end clone of the Paperpillar website built using HTML, CSS, JavaScript, and jQuery. It replicates the layout and design of a creative agency portfolio website.

## Features
- Responsive website layout  
- Interactive navigation using jQuery  
- Portfolio and services sections  
- Smooth scrolling and UI effects  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- jQuery  

## Purpose
This project was created for practice to improve front-end development skills and to understand modern website layouts and interactions
